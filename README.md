# Django Work Project

Учебный проект: инициализация Django с двумя приложениями.

## Приложения
- `workplaces` — рабочие места
- `employees` — сотрудники

## Что сделано
- Создан Django-проект
- Созданы и зарегистрированы приложения в `INSTALLED_APPS`
- Сохранены зависимости в `requirements.txt`
